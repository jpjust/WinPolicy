Segurança do programa

Definindo uma senha

Logo ao iniciar o WinPolicy pela primeira vez, é extremamente recomendado configurar uma senha para o programa. Caso contrário, outros usuários do computador poderão executar o WinPolicy e alterar as configurações e até mesmo configurar uma senha própria, de tal forma que apenas este usuário possa configurar o WinPolicy novamente.

Para configurar uma senha, clique na seção "Segurança". Digite a senha nos campos da janela que aparece e em seguida clique no botão "Salvar senha". Uma mensagem avisando que a senha foi alterada será exibida na tela (e também um pedido para criar um disco de recuperação. Veja como criar o disco mais abaixo). Ao ver a mensagem, significa que a senha foi definida com sucesso.

Caso queira alterar a senha, basta efetuar o mesmo procedimento. Se quiser remover a senha atual deixando o WinPolicy sem senha, clique no botão "Apagar senha".

OBS.: Apagando a senha outros usuários poderão definir uma nova senha e assumir o controle do WinPolicy.

Esta senha será necessária para abrir o WinPolicy e também para desbloquear o Windows quando o AutoLock for configurado para efetuar o bloqueio (o nome de usuário admin deverá ser usado para inserir a senha do WinPolicy no AutoLock).

Após ter configurado uma senha, recomendamos criar um disco de recuperação.

Criando um disco de recuperação

O disco de recuperação do WinPolicy é muito útil em caso de perda de senha. Recebemos muitos e-mails de usuários que perdem suas senhas e não conseguem mais entrar no programa. Caso você tenha esquecido a senha, poderá utilizar o disco de recuperação para acessar o WinPolicy ou desbloquear o sistema.

Para criar um disco de recuperação, entre na seção "Segurança". Em seguida, insira um disco vazio na unidade A: e clique no botão "Disco de Recuperação". O WinPolicy irá solicitar o disco e em seguida irá copiar a senha codificada para o disquete. Após ter copiado a senha, uma mensagem será exibida avisando que a criação do disco obteve sucesso.

O disco agora já está criado. Guarde-o em um local seguro.

NOTA: Este disquete irá funcionar apenas no computador onde foi criado ou em um computador que tenha a mesma senha que a sua. Caso tenha mais de um computador com senhas diferentes, um disco para cada senha distinta deverá ser criado.

Utilizando o disco de recuperação

Em caso de perda de senha, o disco de recuperação poderá ser utilizado. Existem duas situações para a utilização do disco:

1ª: Você perdeu a senha e precisa acessar o WinPolicy.

Execute o WinPolicy. Entre na seção "Segurança" e em seguida clique no botão "Disco de Recuperação". Uma mensagem solicitando o disco será mostrada. Insira o disco na unidade A: e clique no botão "OK". O WinPolicy irá comparar a senha do programa com a senha do disquete, se forem iguais, ele irá liberar o acesso. Caso a senha seja diferente, o programa continuará bloqueado.

2ª: Você perdeu a senha e o sistema está com o acesso bloqueado.

Insira o disquete com a senha na unidade A:. Na tela de bloqueio do AutoLock, clique no botão "Disco de Recuperação". O WinPolicy irá comparar a senha do programa com a senha do disquete, se forem iguais, ele irá liberar o acesso. Caso a senha seja diferente, o sistema continuará bloqueado.

NOTA: Para saber como bloquear o sistema, visite o tópico Bloqueio.
