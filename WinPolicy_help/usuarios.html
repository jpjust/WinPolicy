Cadastro de usuários

O WinPolicy permite que você cadastre outros usuários que poderão desbloquear o Windows ou desligar o computador caso ele esteja bloqueado, mas que não poderão alterar as configurações do programa. Abaixo, as instruções como cadastrar e descadastrar usuários. Todas as operações são feitas na seção "Usuários".

NOTA: Para bloquear o acesso ao computador e usar o cadastro de usuários para restringir a liberação do sistema, você precisa ativar o AutoLock. Verifique a seção "Bloqueio" para obter mais informações.

Cadastrando usuários

Para adicionar um usuário, basta digitar o nome na caixa de texto "Novo usuário:" e em seguida clicar no botão "Adicionar". O nome do usuário adicionado irá para a lista de usuários.

Após ter adicionado um usuário, você deve definir uma senha. Para fazer isto, selecione o usuário na lista "Usuário:" e em seguida clique no botão "Alterar Senha".

Definindo horários de acesso

Para definir os horários que um usuário poderá usar o computador, selecione o nome do usuário na lista "Usuário:". Em seguida, marque os dias da semana que ele tem acesso e adicione os intervalos de horário nas duas caixas de texto mais abaixo. Após ter preenchido as informações, clique no botão "Adicionar" que fica abaixo da lista de horários permitidos.

O usuário poderá usar o computador nos dias marcados e nos horários listados. Não é preciso mais nada para salvar as configurações do usuário, esses dados são salvos imediatamente após ter selecionado as opções.

Excluindo um usuário

Para descadastrar um usuário, basta selecionar seu nome na lista de usuários e clicar no botão "Excluir".
