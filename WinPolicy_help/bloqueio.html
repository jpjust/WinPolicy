Bloqueio do sistema

Clicando na seção "Bloqueio" você pode configurar o WinPolicy para bloquear o Windows na inicialização, no logon ou em horários programados, além de ativar outros recursos.

Todos estes recursos só poderão ser utilizados se o utilitário AutoLock do WinPolicy estiver sendo executado, exceto as opções "Bloquear automaticamente ao iniciar o Windows" e "Bloquear o computador após o logon". Para ativar o AutoLock, basta marcar a opção "Ativar o AutoLock".

Você pode também restringir o bloqueio manual do sistema pelo AutoLock. Basta marcar a opção "Solicitar senha para bloquear o computador pelo AutoLock".

NOTA: Verifique a seção "Cadastro de usuários" para obter mais informações sobre a liberação de acesso ao computador após o bloqueio.

Bloqueando na inicialização

Para bloquear o Windows logo na sua inicialização, marque a opção "Bloquear automaticamente ao iniciar o Windows". Sempre que o computador for ligado, o WinPolicy pedirá uma senha para liberar o acesso. Apenas o administrador do WinPolicy e os usuários cadastrados poderão liberar o acesso (Esta opção só tem efeito nos Windows 95, 98 e Me).

Bloqueando após o logon

Também é possível bloquear o acesso ao Windows imediatamente após alguém ter efetuado o logon no sistema. Para isto, basta marcar a opção "Bloquear o computador após o logon".

Bloqueando em horários determinados

Você pode também bloquear o Windows em horários determinados. Insira os horários de bloqueio na caixa de texto abaixo das opções e em seguida clique no botão "Adiconar". A hora deve estar no formato HH:MM, por exemplo: 08:00, 10:30, 17:45, etc. Quando o relógio do sistema chegar em um dos horários programados, o AutoLock irá bloquear o Windows imediatamente.

Se quiser remover um horário, basta selecioná-lo na lista e clicar no botão "Remover".

Desativando teclas de sistema

O WinPolicy permite também desativar as combinações de teclas CTRL+ALT+DEL, ALT+TAB e CTRL+ESC. Para fazer isto, basta marcar a opção "Desativar teclas do sistema".
