Parâmetros de linha de comando

Você pode fazer algumas operações no WinPolicy através de linha de comando, como por exemplo, carregar um perfil. Abaixo, uma tabela com os parâmetros de linha de comando:

/load "arquivo.wpp" Carrega um arquivo de perfil no WinPolicy e aplica suas restrições.

Exemplo: "C:\Arquivos de Programas\Justsoft WinPolicy\WinPolicy.exe /load C:\perfil.wpp"
/loadusers Usado com o /load, carrega também as configurações de usuários, caso estejam salvas no arquivo de perfil.
/allusers Usado com o /load, carrega as restrições do arquivo de perfil e aplica em todos os usuários do Windows.
/silent Usado com o /load, efetua todas as operações da linha de comando sem exibir mensagens de notificação.
/pwd senha Usado com o /load, utiliza a senha especificada na linha de comando para aplicar o perfil. ATENÇÃO: Este parâmetro foi adicionado a pedido de alguns usuários. O uso dele em arquivos de lotes (.bat) pode permitir que outros usuários do computador vejam a senha do WinPolicy.

Exemplo, a senha é "xyz": "C:\Arquivos de Programas\Justsoft WinPolicy\WinPolicy.exe /load C:\perfil.wpp /pwd xyz"
/setpwd Abre uma janela de texto para alteração da senha do usuário "admin".
/setpwdf Abre uma janela de texto para alteração da senha do usuário "admin". Se uma senha já tiver sido configurada, nada acontecerá.

Exemplo com todos os parâmetros: caso deseje carregar o perfil em "C:\Grupo1.wpp", carregando também os dados dos usuários cadastrados no arquivo de perfil, aplicando as restrições em todos os usuários do Windows, sem mostrar nenhuma caixa de confirmação e especificando a senha pela linha de comando:

"C:\Arquivos de Programas\Justsoft WinPolicy\WinPolicy.exe" /load "C:\Grupo1.wpp" /loadusers /allusers /silent /pwd biscoito

No exemplo acima, a senha do WinPolicy é "biscoito".
